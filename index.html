System Setup

  • For git guidelines and instructions, refer to Git readme*

Install necessary packages

On MAC OS X

brew install git
brew install wget

On Ubuntu

apt install git
apt install wget

Install NVM and node version 10

touch ~/.bash_profile
wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.32.1/install.sh | bash
source ~/.bash_profile
nvm install 10
nvm use 10
nvm alias default 10

Install global packages

npm install -g loopback-cli yarn nodemon

Install local dependencies from project directory

npm install or yarn install

Creating a local postgresql database

Install postgresql server locally. Create a user named admin password as root Create a database named boilerplate-database

Starting server on your machine

You can start forever with watch flag, which will reload your server on any file change. You can access the application at localhost:3000 and the swagger ui explorer at localhost:3000/explorer

  • Starting Node server nodemon .

  • Starting Node Console: node . --console

Text editor settings

Please make sure your text editor is configured properly before you start contributing. Use below settings for sublime text editor.

"tab_size": 2,
"translate_tabs_to_spaces": true,
"trim_trailing_white_space_on_save": true,
"ensure_newline_at_eof_on_save": true,

Use below settings for visual studio Code. Install tslint, HTMLHint, stylelint, stylefmt and eslint pluggins for VSCode.

"editor.tabSize": 2,
"editor.insertSpaces": true,
"editor.trimAutoWhitespace": true,
"files.insertFinalNewline": true,
"editor.formatOnSave": true
